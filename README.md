## ⚡️ Installation & Lancement

1. **Cloner le dépôt** :
```bash
 git clone https://github.com/Rayanthedevloop/mon-portfolio.git
 cd mon-portfolio
```

2. **Installer les dépendances** :
```bash
 npm install
```

3. **Lancer le projet** :
```bash
 npm run dev
```

4. **Ouvrir dans le navigateur** :
```
 http://localhost:5173
```
